<objective>
Replace blocking time.sleep() calls in BmcuSerial.connect() and BmcuFeeder._cmd_enable()
with event-driven BOOT message detection and structured ENABLE retry logic.

Purpose: Reduce Klipper startup delay from ~5s worst-case to however long firmware
actually takes to boot. Eliminate unnecessary 5-second sleep in manual BMCU_ENABLE command.

Output: Modified bmcu_feeder.py with BOOT wait + ENABLE retry, updated MockSerial with
readline queue, 6 new unit tests covering all handshake paths.
</objective>

          - Each iteration calls s.readline(), decodes with 'ascii' errors='replace', strips.

          - MAX_ENABLE_ATTEMPTS = 3, ENABLE_RETRY_DELAY = 2.0 (per D-05).
          - Each attempt: s.write(b"ENABLE\n"), resp = s.readline() decoded/stripped.
          - Log each attempt with logger.info("BMCU: ENABLE attempt %d response: %s").
          - Success: resp.startswith("ENABLE ok") — covers both "ENABLE ok fil=..." and "ENABLE ok already".
          - Failure: log warning, _time.sleep(ENABLE_RETRY_DELAY) between attempts.
          - Use for/else: else clause raises Exception per D-06 with message "BMCU: ENABLE handshake failed after %d attempts — check wiring, firmware, and power".

       d. After successful ENABLE: s.timeout = 0, self._serial = s, register_fd — same as current code (lines 72-77), unchanged.

       - After the self._serial is None guard:

         - Call self._serial.send("ENABLE\n").
         - Read response: self._serial._serial.readline().decode('ascii', errors='replace').strip().
         - Restore self._serial._serial.timeout = 0.
         - Call gcmd.respond_info("BMCU: ENABLE response: %s" % resp).

    Critical invariants to preserve:
    - s.dtr = True; s.rts = False must remain immediately after s.open() with no intervening calls.
    - All log lines use logger.info/logger.warning with % formatting (not f-strings).
    - All decode calls use .decode('ascii', errors='replace').strip().
    - Exception from connect() propagates through _handle_connect() into printer.config_error().
